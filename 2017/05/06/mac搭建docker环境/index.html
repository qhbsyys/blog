mac搭建docker环境

发表于 2017-05-06

Docker依赖于LXC(Linux Container)，可以直接下载配置好的Linux镜像，非常容易就搭建好了。只需要简单的三步:

1.安装VirtualBox
2.安装boot2docker
3.初始化

运行LXC（这里用的是一个Tiny Core Linux，完全在内存中运行，只有24MB，启动时间小于5秒的boot2docker）虚拟机需要安装一个virtual box，VB只需要安装好，不用运行就可以。

搭建环境

  1. 安装VirtualBox。安装完之后不需要创建新虚拟机。

  2. 安装boot2docker

  3. 配置和初始化

其实运行 boot2docker init时会自动下载boot2docker.iso，但是一般下载很慢，不如自己事先下载好。

启动和使用

  1. boot2docker up 启动服务启动后可以在VirtualBox中看到一个名为boot2docker-vm的虚拟机处于运行状态:
  2. boot2docker ssh 连接到虚拟机。如果需要密码，尝试一下docker。

附上boot2docker命令:

cmd func
init Create a new Boot2Docker VM.
up | start | boot Start VM from any states.
ssh [ssh-command] Login to VM via SSH.
save | Suspend Suspend VM and save state to disk.
down | stop | halt Gracefully shutdown the VM.
restart Gracefully reboot the VM.
poweroff Forcefully power off the VM (may corrupt disk image).
reset Forcefully power cycle the VM (may corrupt disk image).
delete | destroy Delete Boot2Docker VM and its disk image.
config | cfg Show selected profile file settings.
info Display detailed information of VM.
ip Display the IP address of the VM’s Host-only network.
shellinit Display the shell commands to set up the Docker client.
status Display current state of VM.
download Download Boot2Docker ISO image.
upgrade Upgrade the Boot2Docker ISO image (restart if running).
version Display version information.
